Tester un effet

Principes généraux

Y-a-t’il un effet de ceci sur cela ?

Effet de ceci sur cela : lien entre ceci et cela à l’échelle d’une population

Exemples :

  • Les poids moyens sont-ils les mêmes pour toutes les variétés ?
  • Le diamètre dépend-il de l’indice de clarté L ?
  • Peut-on prédire la maturité par les indices de colorimétrie ?

Dans ces deux problématiques, que sont

  • la population visée ?
  • ceci (variable explicative) ?
  • cela (variable à expliquer) ?

Effet groupe

Exemple

Les poids moyens des fruits par variété sont-il différents ?

tab %>% mutate(Mean=round(Mean,digits=2)) %>% t() %>%

Focus sur la comparaison de deux moyennes

 

Les poids moyens des fruits de variétés A et B sont-ils significativement différents ?

 

Alternative : test de Student

  • Test de référence pour des hypothèses nulles du type un paramètre est égal à une valeur cible
  • Ici, l’hypothèse nulle est : la différence entre les poids moyens des fruits de variétés A et B est égale à 0

Test de Student

fruit2 <- fruit %>% filter(Variete%in%c("A","B")) %>% droplevels()
t.test(Poids~Variete,data=fruit2,var.equal=TRUE)

Effet linéaire

Exemple

fruit %>% ggplot() + aes(x=L,y=Diam) + 
  geom_point() + geom_smooth(method='lm') +
  ggtitle("Relation entre diamètre et clarté") + xlab("L") + ylab("Diamètre")

Corrélation

Le coefficient de corrélation \(-1 \leq r \leq 1\) mesure l’intensité du lien linéaire entre deux variables :

cor(fruit$L,fruit$Diam)
[1] 0.535

Interprétation (à confirmer par graphique) :

  • Si \(r \approx 1\) : relation linéaire croissante très forte
  • Si \(r \approx -1\) : relation linéaire décroissante très forte
  • Si \(r \approx 0\) : absence de lien linéaire
